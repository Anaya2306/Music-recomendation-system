# Music-recomendation-system
Exploratory data analysis on Spotify music features to identify trends and patterns in popularity, energy, tempo, and genres.

## Objectives
- Analyze relationships between audio features and popularity
- Visualize trends across different music attributes

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Insights
- Popular songs tend to have higher energy levels
- Certain tempos correlate with higher popularity

## How to Use
Open notebooks/analysis.ipynb to view the analysis.
